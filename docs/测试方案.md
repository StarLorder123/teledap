## 1. 测试目标与范围

### 1.1 测试目标

验证 TeleDAP 项目各层模块的功能正确性、状态机安全性、协议兼容性以及端到端集成可用性，确保：

1. **DAP 协议解析正确** — Content-Length 帧编码/解码符合 DAP 规范，处理黏包、断包、超长帧等边界情况
2. **进程生命周期安全** — codelldb 子进程的启动、通信、关闭全过程无泄漏、无死锁
3. **状态机安全** — `Disconnected → Connected → Initialized → Running ↔ Halted` 状态转换合法，操作在非法状态下被正确拒绝
4. **MCP 工具调度正确** — 24 个工具的状态感知过滤、参数反序列化、返回值格式均正确
5. **端到端可用** — AI 客户端能通过 MCP 协议完成完整的"启动→下断→调试→查变量→关闭"工作流

### 1.2 测试范围

| 模块             | 单元测试 | 集成测试 | 端到端测试 |
| ---------------- | :------: | :------: | :--------: |

| `teledap` (root) |    —     |    —     |     ✅      |

### 1.3 不在此次测试范围

- OpenOCD 集成（Phase 4 未实现）
- 性能压力测试

### 3.2 dap-codec — DAP 帧编解码

| 编号  | 测试用例                                            | 输入                                         | 预期结果                                        |

| DC-05 | 多帧黏包 — 一个 buffer 含 2 帧                      | 帧1 完整 + 帧2 完整                          | 两次 decode 各得一条消息                        |
| DC-06 | 多帧黏包 — 帧1 完整 + 帧2 部分                      | 完整帧 + 半帧                                | 第一次得到帧1，第二次 `None`，补充数据后得到帧2 |

### 8.3 测试辅助脚本模板

#### 端到端 MCP 测试脚本 (`test_mcp_e2e.sh`)

```bash
#!/bin/bash
# test_mcp_e2e.sh — TeleDAP MCP Server 端到端测试
set -e

TELEDAP="cargo run --quiet --"
COUNTER=0

send() {
    COUNTER=$((COUNTER + 1))
    echo "{\"jsonrpc\":\"2.0\",\"id\":$COUNTER,\"method\":\"$1\",\"params\":$2}"
}

# 构建请求序列
{
    # 1. initialize
    send "initialize" '{"protocolVersion":"2024-11-05","capabilities":{},"clientInfo":{"name":"e2e-test","version":"1.0"}}'
    
    # 2. 发送 initialized 通知
    echo '{"jsonrpc":"2.0","method":"notifications/initialized"}'
    
    # 3. tools/list
    send "tools/list" '{}'
    
    # 4. tools/call start
    send "tools/call" '{"name":"start","arguments":{"codelldbPath":"codelldb"}}'
    
    # 5. tools/call initialize
    send "tools/call" '{"name":"initialize","arguments":{}}'
    
    # 6. tools/call get_state
    send "tools/call" '{"name":"get_state","arguments":{}}'
    
    # 7. tools/call shutdown
    send "tools/call" '{"name":"shutdown","arguments":{}}'
    
} | $TELEDAP 2>/dev/null | while read -r line; do
    echo "RESPONSE: $line" | head -c 200
    echo
done

echo "E2E test completed with $COUNTER requests."
```

#### Windows PowerShell 端到端测试 (`test_mcp_e2e.ps1`)

```powershell
# test_mcp_e2e.ps1 — TeleDAP MCP Server E2E test (Windows)
$ErrorActionPreference = "Stop"

$requests = @(
    '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2024-11-05","capabilities":{},"clientInfo":{"name":"e2e-test","version":"1.0"}}}'
    '{"jsonrpc":"2.0","method":"notifications/initialized"}'
    '{"jsonrpc":"2.0","id":2,"method":"tools/list","params":{}}'
    '{"jsonrpc":"2.0","id":3,"method":"tools/call","params":{"name":"get_state","arguments":{}}}'
)

$input = $requests -join "`n"
$input | cargo run --quiet 2>$null | ForEach-Object {
    Write-Host "RESPONSE: $($_.Substring(0, [Math]::Min(200, $_.Length)))"
}

Write-Host "E2E test completed."
```
